Chat

messaging app for effortless engagement.
My Role
UI Designer
Duration
2 weeks
Tools
Figma, Adobe Illustrator, Adobe Photoshop
Overview
CHAT, a UI Design Course Project, is a mobile messaging app that prioritizes convenience and an attractive interface. The primary objective was to create a seamless user experience by skiping the UX process. By applying UI design principles, the app was designed to feel intuitive, enabling users to effortlessly initiate conversations, add contacts, and join group chats. The emphasis was on reducing the learning curve and enhancing user engagement.
Objective
My goal for this project was to create a communication app that provided users with a sense of familiarity, making it easy to start conversations, add contacts, and join group chats. To expedite the design process, I leveraged UI toolkits and reused existing UI elements for certain sections of the CHAT screens. For instance, the search and keyboard components were sourced from the Apple iOS 14 UI toolkit. However, I made an effort to use these existing elements sparingly, aiming to give each design a distinct look and feel.
Design Process
Wireframe
To prioritize the design aspect of the project, I chose to skip the UX research process and directly moved into the design stages. Using Figma, I created mid-fidelity wireframes as a foundation for the app's interface. Drawing from my experience with messaging apps, I incorporated established design principles and user flows. I then refined the designs into high-fidelity wireframes, focusing on visual details and user-friendly elements. The goal was to create an aesthetically pleasing and intuitive app experience.

Prototype
After finalizing the screen designs, I created professional mockups in Figma to showcase the CHAT app. These mockups included static presentations for different iPhone displays and dynamic mockups of the app in action. These visuals provide a comprehensive overview of the app's design and functionality, making it easier for users to understand its visual appeal and user experience.
Testing
During the testing phase of my UI Design Course Project, I gathered a group of participants who were familiar with the respective operating systems. They explored the app, focusing on navigation, layout, and visual design. The feedback was mostly positive, confirming that the app provided a seamless user experience. It was intuitive, allowing users to easily start conversations, add contacts, and join group chats, enhancing user engagement and minimizing the learning curve.
Reflection
My main goal for this project was to develop a messaging app that offered an intuitive and user-friendly experience. By making use of UI toolkits and existing UI components, I was able to successfully accomplish this objective with the CHAT app. Users who are familiar with messaging apps will find it effortless to navigate and utilize the app's features. By leveraging pre-existing UI elements and toolkits, I streamlined the design process and saved valuable time. I am extremely pleased with the final outcome, as I believe I have created a set of top-notch screen designs.
